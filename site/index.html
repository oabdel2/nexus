The Problem

The $2.5 Trillion Problem

AI inference spending is exploding. Most of it is wasted on over-provisioned models for simple tasks.

Every request hits GPT-4

Most apps use premium models for everything, even trivial queries like “What’s 2+2?” — burning budget on tasks a tiny model handles perfectly.

No quality verification

Gateways route blindly. Nobody measures if the cheap model’s answer was actually good enough — so you either overspend or gamble on quality.

Zero learning

Static routing rules never improve. Your gateway treats request #1 the same as request #1,000,000 — no feedback, no confidence map, no evolution.

Features

Everything You Need for Intelligent Routing

Four pillars that make Nexus the smartest AI gateway you can self-host.

🧠

Intelligent Routing

A hybrid TF-IDF classifier scores every request in 11μs using 4 signals. Cascade routing tries the cheap model first and escalates only when confidence drops below threshold.

  • 68-example TF-IDF with online learning
  • Hybrid scoring: TF-IDF 50% + keywords 25% + structural 15% + length 10%
  • Cascade routing with 0.78 confidence threshold
  • 6-signal confidence scoring
⚡

Performance

7-layer caching from sub-millisecond exact match to semantic similarity with BGE-M3 embeddings. Prompt compression saves 20-35% on tokens before they hit the model.

  • L1 Exact + L2 BM25 + L2 Semantic cache layers
  • Reranker verification & synonym learning
  • 4 prompt compression strategies (20-35% savings)
  • Model warmup & connection pooling
🔒

Enterprise Security

12+ security layers from network to application. OIDC SSO, RBAC, prompt injection guard with 16 patterns, and full audit logging — all built-in, zero external deps.

  • OIDC SSO, RBAC, API key auth
  • Prompt injection guard (16 patterns)
  • Rate limiting, IP allowlist, CORS, HSTS
  • Error sanitization & webhook HMAC verification
📊

Observability

Real-time cost dashboard, A/B testing with statistical significance, shadow evaluation that compares models in parallel — so you can prove savings without guessing.

  • Cost savings dashboard with real-time charts
  • A/B testing (z-test + Welch's t-test)
  • Shadow evaluation on 10% of traffic
  • Prometheus + Grafana integration
