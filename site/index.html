Quickstart

Up and Running in 60 Seconds

Choose your preferred installation method and start saving.

# 1. Clone the repo $ git clone https://github.com/nexus-gateway/nexus.git $ cd nexus # 2. Configure your providers in .env $ cp .env.example .env $ nano .env # OPENAI_API_KEY=sk-... # ANTHROPIC_API_KEY=sk-ant-... # 3. Launch with Docker Compose $ docker compose up -d # ✓ Nexus is running on http://localhost:8080 # ✓ Prometheus metrics on :9090
# 1. Download the latest release $ curl -sSL https://get.nexus-gateway.dev | sh # 2. Create a minimal config $ cat << EOF > config.yaml server: port: 8080 providers: - name: openai api_key: ${OPENAI_API_KEY} - name: anthropic api_key: ${ANTHROPIC_API_KEY} tiers: economy: gpt-4o-mini standard: gpt-4o premium: claude-sonnet-4 EOF # 3. Run Nexus $ nexus serve --config config.yaml # ✓ Nexus is running on http://localhost:8080
# 1. Install the Python SDK $ pip install nexus-gateway # 2. Use it in your code (3 lines!) from nexus import NexusClient client = NexusClient(base_url="http://localhost:8080") response = client.chat( messages=[{"role": "user", "content": "Explain quantum computing"}], workflow_id="research-bot", step=1 ) print(response.content) # → Routed to 'premium' tier (complexity: 0.82) # → Cost: $0.003 (saved $0.027 vs always-GPT-4)
