Integration Flow

Up and Running in 5 Steps

From zero to optimized AI inference in minutes, not weeks.

🚀
Step 1
Deploy (1 command)
Spin up Nexus with a single command. Docker Compose brings up the gateway, cache, and monitoring stack together.
# Docker Compose (recommended) docker compose up # Or Kubernetes via Helm helm install nexus deploy/helm/nexus/
🔌
Step 2
Connect Providers
Add your API keys. Nexus supports OpenAI, Anthropic, local models via Ollama, and more.
providers: - name: openai api_key: ${OPENAI_API_KEY} - name: anthropic api_key: ${ANTHROPIC_API_KEY}
🎯
Step 3
Point Your App (1 line change)
Change the base URL in your existing code. That's it — the API is fully OpenAI-compatible.
# Python — one line change client = OpenAI(base_url="https://nexus.company.com/v1")
⚡
Step 4
Automatic Value Kicks In
Every request is automatically analyzed, cached, and routed to the optimal model — no configuration needed.
Request → Cache Check → Route → Provider → Cache Store → Response
📊
Step 5
Monitor & Save
Open Grafana. See 50–70% cost reduction. Track every request, cache hit, and model decision in real-time dashboards.
