🔗 Supply Chain Security

The best vulnerability is the one that can't exist. Nexus eliminates entire categories of supply chain attacks by design.

📦

Single Dependency

Nexus depends only on gopkg.in/yaml.v3 beyond Go's standard library. That's it. No transitive dependency trees, no hidden attack vectors.

⚙️

Compiled Binary

Go compiles to a static binary. No runtime dependency resolution, no dynamic imports, no pip install or npm install at deploy time.

🔒

No Supply Chain Attack Vector

With no package manager running in production, there's no mechanism for dependency confusion, typosquatting, or malicious package injection.

🛡️ Runtime Security

Every request passes through 15+ security middleware layers before reaching your AI providers. Defense in depth, not defense by hope.

Prompt Injection Guard

16 attack patterns detected and blocked in real time, including jailbreak attempts, system prompt extraction, and role-play exploits.

RBAC & OIDC SSO

Role-based access control with wildcard permissions. Integrate with any OIDC-compliant identity provider for single sign-on.

Rate Limiting & IP Allowlisting

Configurable per-key and per-IP rate limits with sliding window counters. IP allowlisting for trusted networks.

mTLS Support

Mutual TLS for service-to-service authentication. Both client and server certificates verified for zero-trust networking.

Error Sanitization

No stack traces, internal paths, or sensitive data ever leak to clients. All errors are sanitized and categorized before returning.

Audit Logging

Every request, routing decision, and security event is logged with structured data for compliance and forensics.

🔐 Data Security

Your prompts, responses, and API keys stay where they belong — in your infrastructure, under your control.

Self-Hosted

Nexus runs entirely in your infrastructure. Your data never passes through third-party proxy servers or analytics pipelines.

  • Deploy on-premises, in your VPC, or air-gapped
  • No external data sharing or aggregation
  • Full control over data residency

No Telemetry

Zero telemetry phoning home. No usage analytics, no crash reporters, no beacons. The binary does exactly what you configure and nothing more.

  • No outbound connections except to your configured providers
  • No hidden analytics endpoints
  • Verifiable — source code is auditable under BSL 1.1

Cryptographic Protections

API keys are hashed with SHA-256 and compared in constant time to prevent timing attacks. Webhooks verified with HMAC signatures.

  • SHA-256 key hashing with constant-time comparison
  • Webhook HMAC verification (Stripe, custom)
  • TLS 1.2+ for all provider connections
