Frequently Asked Questions

What counts as a request?

A request is any API call to the Nexus gateway — including chat completions, embeddings, and any proxied call. Cached responses that are served from Nexus’s 7-layer cache still count as requests, but they don’t incur any upstream model cost.

Can I self-host Nexus?

Absolutely. Nexus is a single Go binary with zero external dependencies (only yaml.v3). You can run it via Docker, Docker Compose, or the standalone binary. Enterprise customers also get a Helm chart for Kubernetes with 14 templates, HPA, and ServiceMonitor.

How does Nexus handle my data?

Nexus runs entirely in your infrastructure. Your prompts and responses never leave your environment. We don’t collect, store, or transmit any request data. The TF-IDF classifier runs locally and the cache is stored in your own infrastructure.

How does cascade routing work?

Cascade routing tries the cheapest suitable model first. If the confidence score (measured by 6 heuristics including hedging, completeness, and structure) falls below the 0.78 threshold, the request is automatically escalated to the next tier. This saves 40-70% on routable traffic.

What is shadow evaluation?

On 10% of requests, Nexus runs both the cheap and premium model in parallel and compares the results. This builds a confidence map over time, so the system can prove that routing to cheaper models doesn’t degrade quality. You get statistical evidence, not guesswork.

Can I use my own API keys?

Yes. Nexus is a proxy — you bring your own API keys for OpenAI, Anthropic, Ollama, or any other provider. Nexus routes to the best model using your existing accounts. There’s no markup on model costs.

What happens if I exceed my request limit?

Requests over your plan limit are still processed but at a reduced priority. You’ll receive a notification at 80% and 100% of your limit. You can upgrade your plan at any time to increase your allocation.

How long does integration take?

Most teams are up and running in under 10 minutes. If you’re using the OpenAI SDK, you just change the base_url parameter. No code changes needed beyond that. Run nexus init and the wizard handles the rest.

Is Nexus open source?

Yes. Nexus is fully open source and available on GitHub. The Free tier includes all core routing features. Paid tiers add advanced capabilities like cascade routing, full caching, A/B testing, and enterprise security features.

What models does Nexus support?

Nexus supports any OpenAI-compatible API including OpenAI (GPT-4o, GPT-4o-mini), Anthropic (Claude), local models via Ollama, and any other provider with an OpenAI-compatible endpoint. The TF-IDF classifier is model-agnostic.
